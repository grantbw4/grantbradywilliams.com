# Grant Williams - Personal Website

A clean, minimalist portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Dark theme with warm tones
- Responsive design
- Static site generation for optimal performance
- Easy to update and maintain

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Deployment

This site is configured for static export and can be deployed to:
- GitHub Pages
- Vercel
- Netlify
- Any static hosting service

## Structure

- `/app` - Next.js app directory with pages
- `/components` - Reusable React components
- `/public` - Static assets (images, etc.)
- `/content` - Content files for projects

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18
